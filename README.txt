Amarire Dev - Personal Portfolio Website
======================================================

Description:
------------
Amarire Dev is a professional Frontend Developer and E-Commerce Expert based in Morocco.
This portfolio website showcases skills, projects, and contact information. It supports
multiple languages including English, Arabic, French, and Tamazight (Tzm).

The website is built with modern web technologies and follows best practices for
responsiveness, SEO, accessibility, and security.

Technologies & Tools Used:
--------------------------
- Frontend: HTML5, CSS3 (custom stylesheets), JavaScript
- Backend: PHP (for contact form processing and sitemap generation)
- Database: MySQL (to store visitor messages)
- SEO: sitemap.xml, robots.txt, meta tags (Open Graph, Twitter Cards), structured data (JSON-LD)
- Fonts: Fredoka for consistent typography
- Multilingual support: Dynamic content translation via JSON + JavaScript
- Hosting environment: Compatible with most shared hosting with PHP 7+

Website Structure:
------------------
- assets/: Images, icons, Open Graph assets
- css/: Stylesheets (header3.css, footer.css, home1.css, etc.)
- js/: JavaScript files for UI interactions and translations
- php/: PHP scripts including form handlers and sitemap generator
- db/: Database connection files
- index.php: Homepage
- contact.php: Contact page
- projects.php: Projects listing page
- translations/: Language JSON files (en, ar, fr, tzm)

SEO & Performance:
------------------
- Optimized meta tags for title, description, keywords, author
- Language alternate links for SEO multi-language support
- Open Graph & Twitter Cards meta for social sharing previews
- JSON-LD structured data describing the person and social profiles
- Sitemap auto-generated via PHP script
- robots.txt configured to allow search engines while protecting sensitive files
- Favicon and Apple Touch Icons for cross-platform support
- Google Analytics integrated for visitor tracking and analysis

Setup Instructions:
-------------------
1. Upload all files and folders to your hosting public directory (e.g., public_html).
2. Configure the database connection in `db/db.php`.
3. Ensure your hosting supports PHP 7+ and MySQL.
4. Run the sitemap generator script (`php/generate-sitemap.php`) periodically or set a cron job.
5. Update Google Search Console and Analytics with your verification and tracking codes.
6. Place the favicon files in `/assets/favicon/`.
7. Keep the translation files updated in `/translations/`.

Security Notes:
---------------
- Protect sensitive files using `.htaccess` or server configuration.
- Always use HTTPS and set HSTS for strong security headers.
- Sanitize all user inputs in PHP to prevent XSS and SQL injection.
- Restrict access to admin or generator tools using authentication.
